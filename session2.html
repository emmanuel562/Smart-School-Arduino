1 What is an Actuator?

An actuator is any device that converts electricity into something physical — something you can see, hear, or feel. In Session 1 you learned about outputs. Actuators are the outputs you will actually wire up and program in this session.

📱
Phone screen
Electricity becomes light you can see
🔊
Speaker
Electricity becomes sound you can hear
🚗
Car engine
Electricity becomes movement
📳
Phone vibration
Electricity becomes movement you can feel
💡
One sentence to remember If it does something physical when you send it electricity, it's an actuator.

2 Digital vs Analog Output

Not all outputs work the same way. Some are simple on/off switches. Others let you control exactly how much power goes out — like a dimmer switch on a lamp.

🔵 Digital — only 2 values
0
1
LOW = OFFHIGH = ON
Like a light switch — fully on or fully off. Use digitalWrite().
🟣 Analog (PWM) — 0 to 255 values
0
128
255
Dim / OffFull brightness
Like a dimmer switch. Use analogWrite() — but only on PWM pins.
~
PWM pins only analogWrite() only works on pins marked with ~ on the Arduino Uno: 3, 5, 6, 9, 10, and 11. Using it on other pins won't give you dimming.

4 External LED — Hands On

Time to wire up your first external LED — not the tiny one on the board, but one you connect yourself on the breadboard.

Component Pin Connect To Notes
Long leg (+) Pin 7 220Ω resistor in series
Short leg (–) GND Direct connection
external_led.ino 
int ledPin = 7;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH);  // LED ON
  delay(500);
  digitalWrite(ledPin, LOW);   // LED OFF
  delay(500);
}

5 Buzzer

The piezo buzzer turns electricity into sound. Instead of just ON/OFF, you control the frequency — how high or low the beep sounds.

Component Pin Connect To Notes
+ (positive) Pin 8 Direct connection
– (negative) GND Direct connection
tone(pin, freq)
Plays a sound at the given frequency. 262 = Middle C, 440 = A4, 1000 = high beep.
noTone(pin)
Stops the sound. Always call this before starting a new tone on the same pin.
buzzer.ino 
int buzzerPin = 8;

void setup() {
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  tone(buzzerPin, 1000);
  delay(500);
  noTone(buzzerPin);
  delay(500);
}

6 Servo Motor

A servo rotates to an exact angle — not just spinning freely like a DC motor. To control it, you need an extra toolkit: the Servo library. Think of #include as borrowing tools someone else already built so you don't have to write everything from scratch.

Wire Colour Connect To Notes
Red 5V Power supply
Brown / Black GND Ground
Orange / Yellow Pin 9 Signal — PWM pin
servo.ino 
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9);
}

void loop() {
  myServo.write(0);
  delay(1000);
  myServo.write(90);
  delay(1000);
  myServo.write(180);
  delay(1000);
}
📐
Angles matter myServo.write(0) goes all the way left. write(90) is the middle. write(180) goes all the way right.

7 RGB LED

An RGB LED has three tiny LEDs inside — red, green, and blue. By mixing different brightness levels with analogWrite, you can create almost any colour.

Leg Connect To Notes
Red leg Pin 9 220Ω resistor in series
GND leg (longest) GND Direct connection
Green leg Pin 10 220Ω resistor in series
Blue leg Pin 11 220Ω resistor in series
rgb_led.ino 
int redPin   = 9;
int greenPin = 10;
int bluePin  = 11;

void setup() {
  pinMode(redPin,   OUTPUT);
  pinMode(greenPin, OUTPUT);
  pinMode(bluePin,  OUTPUT);
}

void loop() {
  // RED
  analogWrite(redPin, 255); analogWrite(greenPin, 0);   analogWrite(bluePin, 0);
  delay(1000);
  // GREEN
  analogWrite(redPin, 0);   analogWrite(greenPin, 255); analogWrite(bluePin, 0);
  delay(1000);
  // YELLOW
  analogWrite(redPin, 255); analogWrite(greenPin, 200); analogWrite(bluePin, 0);
  delay(1000);
  // BLUE
  analogWrite(redPin, 0);   analogWrite(greenPin, 0);   analogWrite(bluePin, 255);
  delay(1000);
}
