💻 Arduino Code Cheat Sheet

cheatsheet.ino 
// ── STRUCTURE ───────────────────────────
void setup() { }   // runs once at startup
void loop()  { }   // runs forever after setup

// ── VARIABLES ───────────────────────────
int   myNumber = 10;    // whole number
float myDecimal = 3.14; // decimal number
bool  isOn = true;       // true or false only

// ── DIGITAL ─────────────────────────────
pinMode(pin, OUTPUT);        // set pin as output
pinMode(pin, INPUT);         // set pin as input
digitalWrite(pin, HIGH);     // turn pin ON  (1)
digitalWrite(pin, LOW);      // turn pin OFF (0)
digitalRead(pin);             // read: returns HIGH or LOW

// ── ANALOG ──────────────────────────────
analogWrite(pin, value);      // PWM output: 0-255 (pins ~3,5,6,9,10,11)
analogRead(pin);              // read: returns 0-1023 (analog pins A0-A5)

// ── TIMING ──────────────────────────────
delay(1000);                  // wait 1000ms = 1 second
millis();                     // ms since board started

// ── SOUND ───────────────────────────────
tone(pin, frequency);         // play tone (Hz)
tone(pin, frequency, duration);// play for duration ms
noTone(pin);                  // stop tone

// ── SERVO ───────────────────────────────
#include <Servo.h>
Servo myServo;
myServo.attach(pin);          // connect servo to pin
myServo.write(angle);         // move to angle (0-180)

// ── SERIAL MONITOR ──────────────────────
Serial.begin(9600);           // start in setup()
Serial.println(value);        // print with newline
Serial.print(value);          // print without newline

// ── LOGIC ───────────────────────────────
if (x > 10) { }              // if condition
if (x > 10) { } else { }    // if / else
for (int i=0; i<10; i++) { } // loop 10 times

🔍 Common Errors & How to Fix Them

❌

'X' was not declared in this scope

You used a variable or function name that doesn't exist or has a typo. Check spelling and capitalisation — Arduino is case-sensitive.

❌

Expected ';' before '}'

You're missing a semicolon at the end of a line. Look at the line before the line number in the error message.

❌

No port selected / Board not found

Check Tools → Port and select the COM port that appeared when you plugged in the Arduino. Try a different USB cable if no port appears.

❌

analogWrite() has no effect

You're using analogWrite() on a pin that doesn't support PWM. Only pins marked with ~ work: 3, 5, 6, 9, 10, 11 on the Uno.

❌

LED doesn't light up

Check: (1) long leg to the pin, short leg to GND, (2) resistor is in the circuit, (3) pin number in code matches actual wire, (4) pin is set to OUTPUT.

❌

Servo twitches but doesn't move properly

The Arduino's 5V pin may not supply enough current. Try powering the servo directly from an external 5V source, or use fewer servos at once.
